1. Home Page — Public Landing

URL: /  |  Audience: General public, prospective customers  |  Goal: Conversion to quote request

Sections (top to bottom)

A. Hero / Header

  • Company logo, tagline "JC ON THE MOVE LLC"
  • Sticky "Login" button (top right)
  • "Get a Free Quote" CTA button → opens booking form / chatbot
  • Phone number link: (906) 285-9312
  • Dedication banner: "In memory of Nature Made Jewls"

B. Services Grid

  • 8 service cards: Moving, Junk Removal, Snow Removal, Cleaning, Handyman, Demo, Flooring, Painting
  • Each card: icon + name, clickable to scroll to pricing or pre-fill chatbot

C. Live Pricing Calculator

  • Toggle: Labor Only vs. Truck Included
  • Crew size slider: 1–5 movers
  • Duration input (hours)
  • ZIP code input → calculates drive distance from Ironwood MI at $40/hr/mover, 45 mph
  • Add-ons: assembly, packing materials, stairs fee, appliances
  • Real-time estimate display: labor, drive time, add-ons, total

D. Testimonials Carousel

  • Rotating approved customer reviews
  • Star rating + reviewer name + text

E. Photo Gallery

  • Grid of job photos from past moves
  • Tap/click to enlarge

F. Booking Chatbot

  • Floating/embedded chat widget
  • Collects: name, phone, email, service type, address, date, details
  • Submits as a lead to backend
Possible Congestion: Hero + Calculator + Chatbot overlap in purpose. Consider: calculator as a quick preview, chatbot as the main lead-capture CTA — not both at equal prominence.

2. Customer Portal

URL: /portal  |  Audience: Logged-in customers  |  Goal: Track jobs, earn & redeem JCMOVES

Tab 1 — My Jobs

  • List of all customer quote/service requests
  • Status badges: Pending, Confirmed, In Progress, Completed
  • Filter tabs: All / Quote / Activated / Completed
  • "+ Request Quote" button → new lead form
  • Each card: service type, address, price, JCMOVES earned badge
  • "Leave Review" button on completed jobs
  • Package/add-ons details if applicable

Tab 2 — Wallet

  • JCMOVES balance (large display)
  • Total Earned / Total Spent stats
  • Transaction History (paginated list)
  • "How to Earn JCMOVES" section:
    • Post a job: +100
    • Completion bonus: +1,500
    • Per dollar spent: +15
    • Leave a review: +150
    • Refer a friend: +2,500
  • Daily Mining widget (up to 3 claims/day)
  • Referral link + code display

Tab 3 — Rewards

  • Link to Rewards Marketplace
  • Loyalty Tier tracker (Bronze → Silver → Gold → VIP)
  • Progress bar toward next tier
  • Tier benefits summary
  • Quantum Spin (lottery wheel) if eligible

Tab 4 — Profile

  • Name, email, phone display/edit
  • Profile photo upload
  • Notification preferences
  • Logout button

Header / Always Visible

  • Loyalty tier badge
  • JCMOVES balance chip
  • Welcome message + user name
Possible Congestion: Wallet tab is dense — Mining widget + Referral section + Transaction history + Earn guide all compete. Consider splitting Mining and Referral into their own sub-tab or collapsible card.
Repetition: "How to Earn JCMOVES" appears on both the Welcome/Onboarding modal and the Wallet tab. Consider keeping only one authoritative location.

3. Crew HQ (Hub)

URL: /hub  |  Audience: Employees + Admins  |  Goal: Job operations, scheduling, crew management

Always-Visible Header

  • Personalized greeting ("Good Morning, [Name]")
  • Weather widget (Ironwood MI + Iron River) with snow alert flag
  • On-Duty / Off-Duty toggle button
  • JCMOVES wallet balance
  • Stats bar: Active Jobs · Total Leads · Completed

Daily Scripture / Inspiration Card

  • Rotating Bible verse or mover wisdom quote (day-of-year indexed)
  • "Claim Daily Reward" button → awards mining tokens (up to once/day, streak tracked)
  • Announcements marquee (team policy reminders)

Operations Tabs (5 tabs)

Repetition: Admin quick-links appear in the Quotes tab (odd location) and also in Control Center. Move admin links to a dedicated "Admin" tab or the Team tab.

5. Secondary / Linked Pages

Lead Detail Page /lead/:id

  • Quote Builder: Itemized order line items, package selection, hours, special items (hot tub, safe, piano, pool table)
  • Crew Assignment: Select crew members from available workers, bonus mover flag, crew size
  • Square Invoice: Build Order button → creates Square catalog order, sends invoice
  • Status Management: Move through statuses (quote → confirmed → in_progress → completed)
  • Address Confirmation: Admin can override from/to addresses
  • JCMOVES Disbursement: Shows token allocation; triggers on completion
  • Photos: Upload before/after/progress photos
  • Notes: Internal quote notes and customer communication log
Congestion: Lead Detail is the heaviest page in the app — it contains order building, crew management, financial tools, communication, photos, and status all on one screen. Consider tabbing the sections (Summary | Order | Crew | Documents | Activity).

Rewards Marketplace /rewards

  • Grid of shop items (physical goods, discounts, services)
  • Filter by category
  • JCMOVES price + USD price on each item
  • Add to cart → checkout with JCMOVES balance

Staking /staking

  • Tier cards: Flexible, Bronze, Silver, Gold, Diamond

  • Active stakes: progress bar, daily rate, pending rewards, claim/unstake buttons
  • Auto-compound toggle per stake
  • Completed stakes history
  • Treasury reward bonus (dynamic)

Mining /mining

  • Daily token claim (up to 3/day)

Snow Removal /snow-removal

Navigation Structure

Feature Repetition

  • "How to Earn JCMOVES" appears in: Onboarding modal, Customer Wallet tab, Rewards Dashboard
  • JCMOVES balance shown in: Hub header, Customer Portal header, Staking page, Mining page
  • Admin links appear in: Hub Quotes tab, Control Center, navbar
  • Job history appears in: Hub Leads tab (collapsed), Hub History section, Admin Pipeline

Suggested Consolidations

  • Hub Leads tab: Move Leaderboard → Team tab. Move History → separate History tab. Keep Leads tab for active job management only.
  • Customer Portal Wallet: Break into "Wallet" (balance + history) and "Earn" (mining + referral + earn guide) sub-sections or cards.
  • Admin Links: Remove from Hub Quotes tab. Keep only in Control Center + top nav.
  • Lead Detail: Add tabs: Summary | Order Builder | Crew & Schedule | Documents & Photos | History & Notes
  • JCMOVES Earn Guide: One canonical location — Wallet tab. Remove from onboarding and rewards dashboard.
