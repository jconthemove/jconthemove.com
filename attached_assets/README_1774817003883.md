# Replit Homepage Handoff

This folder contains the files needed to move the current local homepage setup into a separate Replit project without copying the entire repo.

## Files included

- `client/src/App.tsx`
- `client/src/pages/home.tsx`
- `client/src/pages/_archive/home.tsx`
- `client/src/components/homepage-booking-calculator.tsx`
- `client/src/components/booking-chatbot.tsx`
- `client/src/components/QuoteForm.tsx`
- `client/src/components/ui/date-picker.tsx`
- `shared/pricing.ts`

## What this gives you

- `/` uses the current marketing homepage from `home.tsx`
- `/legacy-home` and `/welcome` preserve the old splash page
- the homepage calculator and quote flow dependencies are bundled here

## Merge order

1. Copy `client/src/pages/home.tsx`
2. Copy `client/src/components/homepage-booking-calculator.tsx`
3. Copy `shared/pricing.ts`
4. Copy `client/src/components/ui/date-picker.tsx`
5. Copy `client/src/pages/_archive/home.tsx` if you want the backup route
6. Merge `client/src/App.tsx` carefully so Replit keeps its existing routes and only adds:
   - the `LegacyHomePage` import
   - `/legacy-home`
   - `/welcome`

## Existing project dependencies expected in Replit

These files assume the Replit project already has:

- the existing `@/components/ui/*` stack
- `@tanstack/react-query`
- `wouter`
- `lucide-react`
- `@/hooks/use-toast`
- `@/lib/queryClient`
